# Участие в разработке документации

Благодарим за интерес к улучшению документации DST Platform! Мы ценим ваш вклад.

## Как можно помочь

### 1. Сообщить об ошибке
- Используйте шаблон для ошибок
- Проверьте, не была ли ошибка уже зарегистрирована
- Предоставьте как можно больше контекста

### 2. Предложить улучшение
- Создайте issue с описанием предложения
- Объясните, почему это улучшение полезно
- Если возможно, предложите конкретные изменения

### 3. Отправить правки напрямую
1. Сделайте fork репозитория
2. Создайте ветку для ваших изменений
3. Внесите правки
4. Создайте Pull Request

## Правила оформления

### Для текста
- Используйте Markdown-разметку
- Следуйте существующему стилю
- Проверяйте орфографию и грамматику
- Добавляйте ссылки на связанные разделы
